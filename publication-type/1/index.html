    Sampling Query Variations for Learning to Rank to Improve Automatic Boolean Query Generation in Systematic Reviews
    Searching medical literature for synthesis in a systematic review is a complex and labour intensive task. In this context, expert searchers construct lengthy Boolean queries. The universe of possible query variations can be massive: a single query can be composed of hundreds of field-restricted search terms/phrases or ontological concepts, each grouped by a logical operator nested to depths of sometimes five or more levels deep.
    Harrisen Scells, Guido Zuccon, Mohamed A. Sharaf, Bevan Koopman
